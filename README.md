# Daily Expense App

## Description
Daily Expense App is an Android application used to record and manage daily expenses.  
Users can enter amount, description, category using Spinner, and payment mode using RadioButtons.  
The app displays all expenses in a ListView and allows users to perform call, SMS, and email actions with confirmation dialog.

## Features
- Add daily expenses
- Category selection using Spinner
- Payment mode selection using RadioButtons
- Display expenses in ListView
- Call, SMS and Email options
- Confirmation dialog box

## Technologies Used
- Java
- XML
- Android Studio

## Project Structure
- Java files: app/src/main/java
- XML layout files: app/src/main/res/layout
- AndroidManifest.xml: app/src/main
